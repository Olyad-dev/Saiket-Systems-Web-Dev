# My Portfolio

This is my personal portfolio website, built using HTML, CSS, and JavaScript. It showcases my skills, projects, and contact information in a modern and responsive design.

---

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices.
- **Interactive Portfolio**: Includes a Lightbox gallery for showcasing projects.
- **Smooth Scrolling**: Navigation links smoothly scroll to the respective sections.
- **Progressive Web App (PWA)**: Can be installed as an app on mobile devices.
- **Customizable**: Easy to update content, styles, and functionality.

---

## How to Use

### 1. **View the Portfolio**
- Open the `index.html` file in any modern web browser.

### 2. **Customize the Content**
- Open `index.html` and update the following sections:
  - **Header**: Update your name and navigation links.
  - **About Section**: Add your bio, skills, and experience.
  - **Projects Section**: Add your projects with descriptions and images.
  - **Contact Section**: Update your email, phone number, and social links.

### 3. **Customize the Styles**
- Modify `css/styles.css` to change colors, fonts, and layout.

### 4. **Add or Replace Images**
- Replace images in the `images/` folder with your own (e.g., profile picture, project screenshots).

### 5. **Deploy the Website**
- Host your portfolio on platforms like:
  - **GitHub Pages**: [Guide](https://pages.github.com/)
  - **Netlify**: [Guide](https://www.netlify.com/)
  - **Vercel**: [Guide](https://vercel.com/)

---

## Dependencies

This project uses the following third-party libraries and plugins:
- **Lightbox**: For image gallery functionality.
- **Smooth Scroll**: For smooth navigation between sections.

---

## Progressive Web App (PWA)

This portfolio is PWA-ready. Users can install it as an app on their devices. To customize the PWA settings:
- Update `site.webmanifest` with your app name, icons, and theme color.
- Ensure the icons (`android-chrome-192x192.png` and `android-chrome-512x512.png`) are present in the root directory.
